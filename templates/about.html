The TIP Manila Chatbot

Interactive • Automated • 24/7 Availability

Executive Summary

Developed to provide prospective students, parents, and alumni with an interactive solution for FAQs regarding TIP Manila’s undergraduate programs.

Objectives

  • Instant undergraduate info.
  • Build AI chatbot.
  • 24/7 user access.

Tech Stack

  • AI Engine: OpenAI API (GPT-4o-mini)
  • Knowledge Base: Custom TIP Manila Dataset
  • Languages: Python, JavaScript, HTML5/CSS3
  • Hosting: OnRender

Scope & Limitations

Limited to undergraduate queries. Cannot process live enrollment data at this time.
